# User Manual: Notion Clone

Welcome to your new workspace! This manual guides you through the core features and new enhancements.

## Getting Started

### Creating Pages
-   Click **New Page** in the sidebar to create a blank document.
-   Use the **Speed Dial** (`CTRL/CMD + K`) to search or quickly create pages.

### using the Editor
The editor is block-based, similar to Notion.
-   **Slash Commands**: Type `/` to open the menu. You can add:
    -   **Basic Blocks**: Text, Headings, Lists, Quotes.
    -   **Media**: Images, Files, Videos (Drag & Drop supported!).
    -   **Databases**: Inline or Full Page tables.
    -   **Ask AI**: Generate content or brainstorm (Beta).
-   **Formatting**: Select text to bold, italicize, or changing colors.
-   **Drag & Drop**: Grab the handle (‚ãÆ‚ãÆ) to move blocks around.

## Organization

### Sidebar
-   **Favorites**: Star important pages to keep them at the top.
-   **Hierarchy**: Create sub-pages infinitely. Drag pages in the sidebar to reorganize.
-   **Trash**: Recover deleted pages from the Trash can.

## Troubleshooting
-   **Navigation Issues**: If a page didn't appear correctly, refresh the sidebar. A fix for hierarchy sync has been applied.
-   **Synced Blocks**: Slash commands in synced blocks are now scoped correctly to avoid double menus.
